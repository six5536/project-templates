# Project Templates

This repository holds project templates. Each template lives on its own branch — check out the branch matching the template you want to use.

## Templates

| Template          | Branch                                                                            |
| ----------------- | --------------------------------------------------------------------------------- |
| Rust + TypeScript | [`rust-typescript`](https://github.com/six5536/project-templates/tree/rust-typescript) |

## Usage

```sh
git clone -b <branch> --single-branch https://github.com/six5536/project-templates.git <your-project>
```
